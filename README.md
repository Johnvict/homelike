# Setup Guide
 - Clone this Repository:
   - ```$ git clone https://github.com/Johnvict/homelike.git```
 - Navigate into app directory:
   - ```$ cd homelike```
 - Install dependencies
   - ```$ npm install```


#

# Running App
## Start app on development
```
$ npm start
```
## Start app on production
```
$ npm run start:prod
```
## Run Test
```
$ npm run test
```
## Run Test With coverage
```
$ npm test -- --coverage
```
